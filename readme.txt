Find WooCommerce products that do not have any shipping information (dimensions and/or weight)

== Description ==

Many plugins that help you estimate shipping costs rely on accurate product information being entered in WooCommerce. This plugin will help you locate the products with missing shipping information.
[](http://coderisk.com/wp/plugin/missing-product-shipping-info-tool/RIPS-SblNykDnYc)

= Features =
- Displays a list of all products with missing weight or dimensions.
- Ignores any products marked as *virtual*, since these products are not shippable.
- Only available for users with the `manage_woocommerce` capability. 

[![RIPS CodeRisk](https://coderisk.com/wp/plugin/missing-product-shipping-info-tool/badge "RIPS CodeRisk")](https://coderisk.com/wp/plugin/missing-product-shipping-info-tool)

== Installation ==

- First make sure the WooCommerce plugin is installed.
- Install this plugin. You will need to have the `manage_woocommerce` capability to use it.
- Visit *Tools* &rarr; *Missing Shipping Info*.
